## Type System

1.  **int can store any integer**

    > **False**. int can store any integer in the range from: -2147483648 to 2147483647

2.  **Order the following from smallest to largest: short, long, int, byte**
    
    > **byte**, **short**, **int**, **long**

3.  **byte is signed**

    > **False**. byte is not signed.

4.  **Between int and uint, which can store higher numbers?**

    > **uint** can store a larger positive number up to 4294967295.

5.  **Which types can store floating-point numbers?**

    > **float**, **double**, and **decimal** are capable of storing floating-point numbers.

6.  **Which type from question 5 can store the largest number?**

    > Out of the three types above, **double** is capable of storing the largest number.

7.  **Which type from question 5 is the most precise?**

    > In this case, **decimal** would be the most precise with 28-29 digits of precision.

8.  **What type is represented by the following: "7"**
    
    > The above is an example of a **string** type.

9.  **Which type is used to store true or false values?**

    > The **bool** type is used to represent boolean values.
