## Exepti's Game

**Did you make a custom exception? Why?**
>I made a custom exception for learning purposes but I guess because the number picked is a technically valid number, `InvalidDataException` did not correctly meet the challenge's needs and although I could have just used `Exception`, it's not best practice.

**If it wasn't required by this challenge, would you have used an exception?**

>Probably not. I could use the existing code within the `try{ } catch { }` blocks to resolve the game without the need to throw a new exception.
