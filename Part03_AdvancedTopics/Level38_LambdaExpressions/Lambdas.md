## Knowledge Check: Lambdas

1. **Lambdas are a special type of method.**

    >**True**. Lambdas are single use concise methods.

2. **You can name lambda expressions.**

    >**False**. They are single use and don't need a name.

3. **Convert this to a lambda:** `bool IsNegative(int x) { return x < 0; }`

    > `x => x < 0;`

4. **Lambda expressions are limited to only one parameter.**

    > **False**. Lambda expressions can have multiple parameters.
